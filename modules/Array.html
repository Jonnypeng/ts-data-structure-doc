mergeIntervals

  • mergeIntervals(intervals: number[][]): number[][]
    • Defined in Array/mergeIntervals.ts:14

    以数组intervals表示若干个区间的集合，其中单个区间为 intervals[i] = [start-i, end-i]。合并所有重叠的区间，并返回一个不重叠的区间数组，该数组恰好覆盖输入中的所有区间

    mergeIntervals([[1, 3], [2, 6], [8, 10], [15, 18]]) //[[1,6],[8,10],[15,18]]
    mergeIntervals([[1,4],[0,2],[3,5]]) //[0,5]
    mergeIntervals([[1, 4], [0, 2], [3, 5]]) //[[0,5]]
    mergeIntervals([[1, 9], [2, 6], [8, 10], [9, 18]]) // [1,18]

    Parameters

    • intervals: number[][]

      表示若干个区间的集合,[start-i, end-i]

    Returns number[][]

    返回一个不重叠的区间数组,此数组合并了所有重叠的区间
